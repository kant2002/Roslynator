# RCS0027: Add newline before binary operator instead of after it

| Property | Value      |
| -------- | ---------- |
| Id       | RCS0027    |
| Category | Formatting |
| Severity | None       |

## Example

### Code with Diagnostic

```csharp
if (x && // RCS0027
    y &&
    z)
{
}
```

### Code with Fix

```csharp
if (x
    && y
    && z)
{
}
```
