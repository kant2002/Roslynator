# RCS0028: Add newline before conditional operator instead of after it

| Property | Value      |
| -------- | ---------- |
| Id       | RCS0028    |
| Category | Formatting |
| Severity | None       |

## Example

### Code with Diagnostic

```csharp
var y = (x) ? // RCS0028
    Foo() :
    Bar();
```

### Code with Fix

```csharp
var y = (x)
    ? Foo()
    : Bar();
```
